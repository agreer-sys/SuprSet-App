# SuprSet Coaching Knowledge Base

This directory contains curated coaching content that powers the AI Coach's responses. All content reflects Alastair's training philosophy and expertise.

## Directory Structure

- **coaching-philosophy/**: Core training principles and approaches
- **exercise-guides/**: Detailed form cues and technique guides for specific exercises
- **training-principles/**: Programming concepts (progressive overload, periodization, etc.)

## Content Format

Each markdown file should follow this structure:

```markdown
---
tags: [tag1, tag2, tag3]
---

# Title

Brief introduction paragraph explaining the concept.

## Key Points

- Point 1
- Point 2
- Point 3

## Common Mistakes

- Mistake 1
- Mistake 2

## Coaching Cues

Specific verbal cues to use during workouts.
```

## Adding New Content

1. Create a markdown file in the appropriate category folder
2. Follow the format above
3. Run the ingestion command: `npm run ingest-knowledge`
4. Test with the AI Coach to ensure responses align with your philosophy
